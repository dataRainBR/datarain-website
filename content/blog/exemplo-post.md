# Como Adicionar um Novo Post

Este é um exemplo de post em Markdown. Você pode usar toda a sintaxe do Markdown para formatar seu conteúdo.

## Formatação Básica

Você pode usar **negrito**, *itálico*, e `código inline`.

### Listas

Listas não ordenadas:
- Item 1
- Item 2
- Item 3

Listas ordenadas:
1. Primeiro item
2. Segundo item
3. Terceiro item

## Código

Blocos de código com syntax highlighting:

```javascript
function exemplo() {
  console.log("Hello, World!");
}
```

```python
def exemplo():
    print("Hello, World!")
```

## Links e Imagens

[Link para o site](https://datarain.com.br)

Para adicionar imagens, coloque-as em `public/content-images/blog/[slug]/` e referencie assim:

```markdown
![Descrição da imagem](/content-images/blog/exemplo-post/imagem.jpg)
```
